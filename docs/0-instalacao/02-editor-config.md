# 📝 Configurando seu Editor (VS Code)

Para uma experiência de alto nível desenvolvendo para Stellar em Rust, seu editor de código precisa estar devidamente configurado com as ferramentas que garantem autocompletar, checagem de erros e suporte a debug.

Recomendamos o **Visual Studio Code** e a instalação das seguintes extensões fundamentais:

1. **[Rust Analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer)**: A ferramenta definitiva para suporte nativo à linguagem Rust (autocompletar inteligente, navegação no código, formatação e verificação de tipagem).
2. **[CodeLLDB](https://marketplace.visualstudio.com/items?itemName=vadimcn.vscode-lldb)**: Ferramenta obrigatória para depuração (*step-through-debugging*). Permite que você acompanhe o fluxo de execução dos seus *smart contracts*.

Para mais informações sobre o suporte do Rust em diversos editores, acesse a [página oficial de ferramentas do Rust](https://www.rust-lang.org/tools).
